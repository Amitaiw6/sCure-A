sCure — UI Style Guide

Design language for the sCure resin-curing touchscreen. Dark-only theme · fixed 800×480 kiosk viewport · font Geist. Values are the source of truth from src/index.css.

Title color rule: screen & dialog titles are white (foreground). The accent cyan/blue (primary) is reserved for interactive accents and active states — never for titles.
