sCure — Design Handoff

Everything a UX/UI designer needs to work on the sCure touchscreen UI.

What's in this folder

sCure-UI.html⭐ The whole app in one file. Double-click to open — no install, no server, works offline. Send just this file.
style-guide.htmlVisual style guide — colors, typography, radius, spacing, components. Double-click to open.
design-tokens.jsonMachine-readable tokens. Import into Figma via the Tokens Studio plugin.
app/ + start scriptsOptional: the multi-file build served locally (needs Node). Same UI as sCure-UI.html.

1 · The interactive app — easiest (no install)

Double-click sCure-UI.html. The entire UI — every screen, all components, fonts and sample data — is inlined into that one file, so it opens straight in your browser with nothing to install. On first open you'll see a short Setup wizard (tap Get Started → name → skip organization); after that you land on the Home screen. You can also just email this single file to anyone.

2 · Style guide & tokens (no setup)

Double-click style-guide.html to view the design language. Designers using Figma can import design-tokens.json with the free Tokens Studio for Figma plugin to get the exact colors, type scale, radius and spacing as Figma styles/variables.

3 · The interactive app — served version (optional, needs Node)

The app/ folder is the same UI as a normal multi-file web build. If you prefer to run it served (e.g. to test deep links), one click does it:

  1. Windows: double-click start-app-windows.cmd
  2. macOS / Linux: double-click start-app-mac-linux.command

Your browser opens at http://localhost:5050 showing the live UI. Keep the small terminal window open while you browse; close it to stop.

Requires Node.js (one-time install from nodejs.org). The launcher uses it to serve the files. No other install needed — it downloads the tiny server on first run.
No Node? Any static server works. With Python installed, open a terminal in the app folder and run python -m http.server 5050, then visit http://localhost:5050.

4 · Tips for exploring

  • The UI is designed for a fixed 800×480 touchscreen; it scales to fit your window.
  • Navigate with the icons in the top bar: history, alerts, settings, network. Tap a material then Start Cure to see the cure-process screen.
  • Dark-only theme. Titles are white; the cyan/blue accent is only for interactive/active states.
