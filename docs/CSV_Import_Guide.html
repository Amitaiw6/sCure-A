S-Cure CSV Import Guide

How to create and import cure programs via CSV files

1. Overview

The S-Cure system allows you to create custom cure programs by importing CSV (Comma-Separated Values) files. Each CSV file defines a sequence of process steps that the machine will execute in order. This guide explains the required format, valid parameters, and common errors to avoid.

2. CSV File Structure

The CSV file must contain a header row followed by one or more data rows. Each row represents one process step.

2.1 Header Format

Step,Process,Temperature,Time,TimerMode,UVIntensity,UVStart,UVRampPercent,CoolingMode

2.2 Column Reference

# Column Description Status
1 Step Step number (1, 2, 3...) Required
2 Process Process type (see Section 3) Required
3 Temperature Target temperature in °C (20–80) Required*
4 Time Duration in minutes (1–120) Required**
5 TimerMode When the timer starts counting Cure/Bleacher
6 UVIntensity UV light intensity in % (5–100) Cure/Bleacher
7 UVStart When UV turns on Cure/Bleacher
8 UVRampPercent Ramp % at which UV turns on Optional
9 CoolingMode Cooling speed: fast, medium, or slow Cooling only

* Temperature is required for Drying, Heating, Cure, Bleacher, and Cooling. Leave empty for Nitrogen.
** Time is required for Drying, Heating, Cure, and Bleacher. Leave empty for Cooling and Nitrogen.

3. Process Types

Process Description Required Fields Notes
Drying Removes moisture from the part using heat Temperature, Time —
Heating Heats the chamber to target temperature Temperature, Time —
Cure UV curing with 405nm LED light Temperature, Time Supports TimerMode, UVIntensity, UVStart
Bleacher Bleaching with 450nm LED light Temperature, Time Supports TimerMode, UVIntensity, UVStart
Cooling Controlled cooling to target temperature Temperature CoolingMode: fast / medium / slow (default: medium)
Nitrogen N₂ purge of the chamber — No temperature or time needed. Runs automatically if N₂ is enabled on the system.

4. Nitrogen (N₂) Purge Rules

Nitrogen purge steps have specific placement rules that must be followed:

N₂ Purge Requirements
  • Maximum 2 Nitrogen steps per program
  • After Nitrogen, the next step must be one of: Heating, Cure, Bleacher, or Cooling
  • A Cooling step is required somewhere after the Nitrogen step to vent the nitrogen from the chamber
  • At least one Heating, Cure, or Bleaching step must follow the N₂ purge
  • If N₂ is not enabled on the system, Nitrogen steps will be skipped automatically during the cure process

7. What Will Be Rejected

The system will not import a CSV file if any of the following errors are found:

Common Rejection Reasons
  • Missing header row — The first row must contain column names including "Step" or "Process"
  • Less than 4 columns — Each data row must have at least 4 comma-separated values
  • Invalid process type — Must be exactly: Drying, Heating, Cure, Bleacher, Cooling, or Nitrogen (case-sensitive)
  • Temperature out of range — Must be between 20°C and 80°C
  • Time out of range — Must be between 1 and 120 minutes
  • Invalid TimerMode — Must be on-ramp or on-target
  • Invalid UVStart — Must be at-start or at-target
  • Invalid CoolingMode — Must be fast, medium, or slow
  • More than 2 Nitrogen steps — Maximum 2 N₂ purge steps per program
  • Nitrogen without process after — A Heating, Cure, or Bleaching step must follow N₂
  • Nitrogen without Cooling — A Cooling step is required to vent the nitrogen
  • Temperature decreases without Cooling — Temperature can only increase between steps unless a Cooling step resets it
  • Empty file or no valid steps — At least one valid data row is required

8. Timer Mode Explained

Mode Behavior
on-target (default) The step timer starts counting only after the chamber reaches the target temperature. The ramp-up time is not included in the step duration.
on-ramp The step timer starts counting immediately when the machine begins heating. The ramp-up time is included in the total step duration.

9. UV Start Mode Explained

Mode Behavior
at-start UV light turns on immediately when the step begins, even during ramp-up.
at-target (default) UV light turns on only after reaching the target temperature.

10. Cooling Mode Explained

Mode Behavior
fast Maximum fan speed for rapid cooling. Use when speed matters more than gradual temperature change.
medium (default) Balanced cooling speed. Recommended for most materials.
slow Gentle cooling for sensitive materials that may warp with rapid temperature change.

11. Quick Checklist

Before importing, verify:
  • File has .csv extension
  • First row is the header with column names
  • Each row has at least 4 columns
  • Process types are spelled correctly (case-sensitive)
  • Temperature is 20–80°C for Heating, Drying, Cure, Bleacher, and Cooling steps
  • Temperature and Time are left empty for Nitrogen steps
  • Time is left empty for Cooling steps
  • Time is between 1–120 minutes for all other steps
  • TimerMode, UVIntensity, and UVStart are only used with Cure or Bleacher
  • CoolingMode is only used with Cooling steps
  • Maximum 2 Nitrogen steps per program
  • After Nitrogen: must have a Heating, Cure, or Bleaching step
  • If using Nitrogen: must have a Cooling step to vent N₂
Important Notes

• Process names are case-sensitive: use "Heating" not "heating" or "HEATING".
• Empty optional fields should be left blank (not "0" or "null") — just leave nothing between the commas.
• The file must have a .csv extension.
• UTF-8 encoding is recommended. Avoid special characters in file names.
• If N₂ is not enabled on the system, Nitrogen steps are automatically skipped during cure.
