S-Cure CSV Import Guide

How to create and import cure programs via CSV files

1. Overview

The S-Cure system allows you to create custom cure programs by importing CSV (Comma-Separated Values) files. Each CSV file defines a sequence of process steps that the machine will execute in order. This guide explains the required format, valid parameters, and common errors to avoid.

2. CSV File Structure

The CSV file must contain a header row followed by one or more data rows. Each row represents one process step.

2.1 Required Columns

Column Name Description Status
1 Step Step number (1, 2, 3...) Required
2 Process Process type (see Section 3) Required
3 Temperature Target temperature in °C (20–80) Required*
4 Intensity UV light intensity in % (0–100) Cure/Bleacher only
5 Time Duration in minutes (1–120) Required
6 CoolingRate Cooling rate in °C/min (1–20) Cooling only

* Temperature is required for Drying, Heating, Cure, and Bleacher. Leave empty for Cooling.

2.2 Optional Columns (Cure & Bleacher only)

Column Name Description Valid Values
7 UvIntensity Additional UV intensity during heating 0–100 (%)
8 TimerMode When the timer starts counting on-ramp or on-target
9 UvStartMode When UV turns on at-start, at-target, or at-ramp-percent
10 UvRampPercent Ramp % at which UV turns on 10–100 (%) — only when UvStartMode = at-ramp-percent

3. Process Types

Process Description Required Fields Light Source
Drying Removes moisture from the part using heat Temperature, Time None
Heating Heats the chamber to target temperature Temperature, Time None
Cure UV curing with 405nm LED light Temperature, Intensity, Time 405nm UV
Bleacher Bleaching with 450nm LED light Temperature, Intensity, Time 450nm Visible
Cooling Controlled cooling at a specified rate Time, CoolingRate None

6. What Will Be Rejected

The system will not import a CSV file if any of the following errors are found:

Common Rejection Reasons
  • Missing header row — The first row must contain column names including "Step" or "Process"
  • Less than 5 columns — Each data row must have at least 5 comma-separated values
  • Invalid process type — Must be exactly: Drying, Heating, Cure, Bleacher, or Cooling (case-sensitive)
  • Temperature out of range — Must be between 20°C and 80°C
  • Intensity on non-Cure/Bleacher step — Drying, Heating, and Cooling must not have intensity values
  • Time out of range — Must be between 1 and 120 minutes
  • Cooling rate out of range — Must be between 1 and 20 °C/min
  • Invalid TimerMode — Must be on-ramp or on-target
  • Invalid UvStartMode — Must be at-start, at-target, or at-ramp-percent
  • UvRampPercent out of range — Must be between 10% and 100%
  • Empty file or no valid steps — At least one valid data row is required
Important Notes

• Process names are case-sensitive: use "Heating" not "heating" or "HEATING".
• Empty optional fields should be left blank (not "0" or "null") — just leave nothing between the commas.
• The file must have a .csv extension.
• UTF-8 encoding is recommended. Avoid special characters in file names.

7. Timer Mode Explained

Mode Behavior
on-target (default) The step timer starts counting only after the chamber reaches the target temperature. The ramp-up time is not included in the step duration.
on-ramp The step timer starts counting immediately when the machine begins heating. The ramp-up time is included in the total step duration.

8. UV Start Mode Explained

Mode Behavior
at-start UV light turns on immediately when the step begins, even during ramp-up.
at-target (default) UV light turns on only after reaching the target temperature.
at-ramp-percent UV light turns on when the temperature reaches a specified percentage of the ramp. Requires UvRampPercent column (e.g., 50 = UV on at 50% of ramp-up).

9. Quick Checklist

Before importing, verify:
  • File has .csv extension
  • First row is the header with column names
  • Each row has at least 5 columns
  • Process types are spelled correctly (case-sensitive)
  • Temperature is 20–80°C for all non-Cooling steps
  • Intensity is only set for Cure and Bleacher steps
  • CoolingRate is only set for Cooling steps
  • Time is between 1–120 minutes
  • Optional columns (7–10) are only used with Cure or Bleacher
