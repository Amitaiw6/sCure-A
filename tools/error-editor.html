sCure - Error & Warning Editor

Edit error descriptions and troubleshooting steps. System fields (code, severity) cannot be changed.

How to use this editor

1. Click "Load errors.json" to open the current error definitions file
2. Edit the Description, Troubleshooting steps, and Support URL for each error
3. Click "Save errors.json" to download the updated file
4. Copy the downloaded file to public/config/errors.json in the project

Error codes are split into two parts: Prefix (E = Critical Error, W = Warning) and Number (unique ID)
